# anndataR: comparison with similar packages

## Functionality testing
Examines which datatypes are supported by which packages in which slots.
Compares anndataR, zellkonverter (R reader) and schard.

## Runtime benchmarks
Generates dummy datasets and checks how fast these packages read from the H5AD file.
Compares anndataR, zellkonverter (R reader and basilisk) and schard.
